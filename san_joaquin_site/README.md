# san_joaquin_site

## Project Description
This "san_joaquin_site" is a website dedicated to showcasing information related to the San Joaquin area. It includes a homepage and a dedicated page for Sampson, providing users with relevant content and resources.

## Project Structure
```
san_joaquin_site
├── css
│   └── styles.css
├── index.html
├── sampson.html
└── README.md
```

## Files Overview

- **css/styles.css**: This file contains the CSS styles for the website. It will define the visual appearance of the HTML elements across the site.

- **index.html**: This file is the main entry point of the website. It typically contains the structure and content for the homepage.

- **sampson.html**: This file contains the structure and content for the Sampson page of the website.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the project folder in your preferred code editor.
3. Open `index.html` in a web browser to view the homepage.
4. Navigate to `sampson.html` for additional content.
